\section{可视化效果}

\subsection{交互参数}

Streamlit界面左侧栏提供如下参数：
\begin{itemize}
  \item 量子比特数：2或3；
  \item 目标态：从全部计算基态中选择；
  \item 最大Grover迭代次数；
  \item 步进动画间隔；
  \item 是否显示statevector表格、Qiskit电路图、理想/带噪声实验对照。
\end{itemize}

主页面按观察顺序组织为：
\begin{enumerate}
  \item 当前步骤的概率幅图和概率图；
  \item 目标态概率随完整迭代次数变化曲线；
  \item Qiskit标准电路图与交互式门操作视图；
  \item 理想概率和带噪声测量频率对照；
  \item 理论公式与过迭代现象解释。
\end{enumerate}
